Project Overview¶

This project uses data collected from Portuguese secondary school students enrolled in a Mathematics course. The goal is to predict whether a student will pass or fail their final exam (G3 >= 10 = Pass) based on demographic, social, and lifestyle features — including alcohol consumption habits.

Dataset: student-mat.csv — 395 students, 33 features Source: https://www.kaggle.com/datasets/uciml/student-alcohol-consumption

Method: Binary Classification (Pass / Fail)
Final Model: Random Forest Classifier
Evaluation Metrics: Accuracy, Precision, Recall, F1-Score, AUC-ROC

Key Question: Can lifestyle and demographic factors. Such as alcohol consumption, study time, and family background. Reliably predict whether a student will pass their math course?

def wrangle(filepath):

    df = pd.read_csv(filepath)

    df['pass'] = (df['G3'] >= 10).astype(int)

#Drop leakage columns

    leakage = ['G1', 'G2', 'G3']
    df = df.drop(columns=leakage)

# Encode binary yes/no columns as 0/1
    binary_cols = ['schoolsup', 'famsup', 'paid', 'activities',
                   'nursery', 'higher', 'internet', 'romantic']
    for col in binary_cols:
        df[col] = (df[col] == 'yes').astype(int)

#Encode sex
    df['sex'] = (df['sex'] == 'M').astype(int)  # 1=Male, 0=Female

#Encode address
    df['address'] = (df['address'] == 'U').astype(int)  # 1=Urban, 0=Rural

#Encode famsize
    df['famsize'] = (df['famsize'] == 'GT3').astype(int)  # 1=>3, 0=<=3

#Encode Pstatus
    df['Pstatus'] = (df['Pstatus'] == 'T').astype(int)  # 1=Together, 0=Apart

#One-hot encode remaining categoricals
    cat_cols = ['school', 'Mjob', 'Fjob', 'reason', 'guardian']
    df = pd.get_dummies(df, columns=cat_cols, drop_first=True)

    return df

7. Conclusion¶

Summary¶

In this project I built a machine learning pipeline to predict whether a Portuguese secondary school student will pass or fail their Mathematics course, based on lifestyle, social, and demographic features.

Pipeline steps:

  1. Wrangling — Dropped leakage columns (G1, G2), encoded categorical features, and created a binary pass/fail target from the final grade G3.
  2. EDA — Found that higher alcohol consumption (especially weekend drinking) correlates with lower pass rates, and that study time and past failures are among the strongest predictors.
  3. Modeling — Compared a majority class baseline, Logistic Regression, and a Random Forest. The Random Forest was selected as the final model, based on accuracy and AUC.
  4. Evaluation — Evaluated on the held out test set using accuracy, F1-score, AUC-ROC, and a confusion matrix. Cross-validation confirmed the model generalizes well.

Key Findings¶

  • Past failures and absences are the strongest predictors of failing.
  • Study time is a strong predictor of passing.
  • Weekend alcohol consumption (Walc) shows a clear negative relationship with pass rate. Students with higher weekend drinking are less likely to pass.
  • Parental education (especially mother's) correlates positively with pass rate. (My favorite prediction feature, found it interesting.)

Limitations¶

  • The dataset is small, which limits the model's generalizability.
